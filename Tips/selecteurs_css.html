Les types de sélecteurs en Css

* : selecteur universel : Permet de sélectionner toutes les balises de la page html

ul + p { } : premier élément après l'élément (ul) indiqué

#container > p { } : prends tous les éléments enfants directs (sous-div)

#container2 ~ p{ } : sélectionne les éléments directement après l'élément indiqué
